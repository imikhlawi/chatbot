# PDF-Dateien

Lege hier **PDF-Dateien** ab (optional).

- **Jupyter-Notebook:** Lädt alle `*.pdf` aus diesem Ordner und baut den FAISS-Index.
- **API/UI:** PDFs werden per Upload (POST `/api/rag/ingest`) hochgeladen; dieser Ordner wird dafür nicht zwingend genutzt.

**Keine PDFs im Repo** – aus Datenschutz- und Urhebergründen werden sie nicht zu GitHub hochgeladen (siehe `.gitignore`).
